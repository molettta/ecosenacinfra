🚀 PRIMEIROS PASSOS
─────────────────────────────────────────────────────────────────────

1. Abra o menu interativo:

   backup

   (pode executar de qualquer diretório!)


2. Ou use os atalhos diretos:

   backup fazer          # Faz backup agora
   backup listar         # Lista backups
   backup agendar        # Configura automático
   backup sync           # Sync com servidor remoto


3. Configure backup automático (recomendado):

   backup agendar


4. Configure sincronização remota (opcional):

   backup sync
